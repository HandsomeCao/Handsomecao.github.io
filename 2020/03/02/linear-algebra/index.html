逐元素积（哈达玛积）

# 向量
a = np.arange(0,9)
b = a[::-1]
c = a * b
print("dot of vector a,b is {}".format(c))

dot of vector a,b is [ 0  7 12 15 16 15 12  7  0]

# 矩阵
a = np.arange(1, 5).reshape(2,2)
b = np.arange(5, 9).reshape(2,2)
c = a * b
print("dot of matrix a,b is {}".format(c))

dot of matrix a,b is [[ 5 12]
 [21 32]]

a = torch.arange(1,5).reshape(2,2)
b = torch.arange(5,9).view(2, 2)

print(a.mul(b))

特殊函数

sigmoid函数

  • 导数
  • 值域在0和1之间，用于二分类
  • 函数具有非常好的对称性
  • 在趋于无穷时，函数值变化小，容易产生梯度消失
import matplotlib.pyplot as plt
%matplotlib inline
def sigmoid(x):
    return 1. / (1.+np.exp(-x))
x = np.arange(-5, 5, 0.2)
y = sigmoid(x)
plt.plot(x,y)
plt.show()

softplus函数

  • 平滑版的ReLu函数
  • 导数:
def softplus(x):
    return np.log(1.+np.exp(x))
x = np.arange(-10, 10, 0.2)
y = softplus(x)
plt.plot(x,y)
plt.show()

伽玛函数

  • 对于整数$n$：$\Gamma(n)=(n-1)!$
  • $\Gamma(x+1)=x\Gamma(x)$
  • 对于$x \in (0,1)$:可以推出重要公式: $\Gamma(\frac{1}{2}) = \sqrt{\pi}$
from scipy.special import gamma
x = np.linspace(0, 5, 100)
y = gamma(x)
plt.plot(x,y)
plt.show()
